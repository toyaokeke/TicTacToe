# Tic Tac Toe Computer

This is a console based tic tac toe game where you can play another person or computer from the command line / terminal

## Requirements

- [Java SDK](https://www.oracle.com/java/technologies/javase-downloads.html) minimum Java 8.x.x

## How to Run

In your terminal run the following:

```shell
javac Game_lab3.java
java Game_lab3
```
And follow the instructions on the screen.

Enjoy! :smile:
